*** Instructions on how to run Boogie of the Deep ***

Requirements:
* Node.js (v18 or newer recommended)
* npm (comes with Node)

# 1. Clone or download the repository
git clone <repo-url>
cd boogie-of-the-deep

# 2. Install dependencies
npm install

# 3. Start the game
npm run dev

# 4. Open browser at:
http://localhost:5173

# 5. Enjoy!
